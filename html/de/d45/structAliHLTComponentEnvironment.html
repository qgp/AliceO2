Detailed Description

This was the original definition of the running environment. Due to a bug in the AliRootWrapperSubscriber/SimpleComponentWrapper, this structure can not be used any longer but is kept for backward compatibility.

Note
The external interface provided by the libHLTbase is now kept frozen but should not be used any more. Use the interface provided by the libHLTinterface library.
